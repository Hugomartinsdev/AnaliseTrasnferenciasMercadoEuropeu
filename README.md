# AnaliseTrasnferenciasMercadoEuropeu
Trabalho criado para a materia de modelagem estatica, onde eu tive que buscar um dataset e fazer o tratamento de dados e busca exploratoria para a criação de um dashboard interetivo.
Foi usado Python para o tratamento e o PowerBi para a criação do dashboard


## Detalhes sobre o dashboard
Na aba/pagina de idade e temporada do dashboard exite um filtro que é o de faixa etária, usada no luga da coluna idade para deixar a escolha melhor do que ter idades de 15 a 43 anos a seguir sera mostrado uma liista de quais idades estão dentro de cada faixa etária
<br>
<ul>
  <li>15 a 24: Tem potencial</li>
  <li>25 a 28: Auge</li>
  <li>29 a 32: Joga em alto nível</li>
  <li>32 em diante: Fim de carreira</li>
</ul>
Essas idades foram escolhidas de acordo com as estatisticas de jogadores e pesquisas sobre analistas onde quando eles consideram que joadores ainda são promesas e quando não são e quando eles começam a perder rendimento pela idade
